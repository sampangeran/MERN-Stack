Process Control and Dynamics

This course dealing with chemical engineering processes controls. This course is about Introduction to characteristics of process control problems and the objectives of process control; Brief review on the design of process control systems; Control system hardware and modelling of dynamic behaviour of chemical processes; Analysis of chemical process dynamics; Analysis and design of feedback control system; Analysis and design of advanced control systems.

References:

  1. Marlin, T. E. (2000). Process Control: Designing Processes and Control Systems for Dynamic Performance (2nd ed.). McGraw-Hill.
  2. Seborg, D. E., Edgar, T. F., Mellichamp, D. A., & Doyle III, F. J. (2017). Process Dynamics and Control (4th ed.). John Wiley & Sons, Inc.
  3. Towler, G. & Sinnott, R. (2013). Chemical Engineering Design: Principles, Practice, and Economics of Plant and Process Design (2nd ed.). Butterworth-Heinemann.

Theoretical Models of Chemical Processes

Process control is defined by the mathematical models. To learn process control, first students must be introduced to:

  1. General Modeling Principles
  2. Degree of Freedom Analysis
  3. Modelling Examples
  4. Linearization
  5. Dynamic Models of Representative Processes

Modeling and Analysis for Process Control

After introduced to the modelling, this session will covers materials in process control modelling, which are:

  1. The Laplace Transform
  2. Input-Output Models and Transfer Functions
  3. Block Diagrams

Dynamic Behaviour

Dynamic behaviour in chemical processes must be treated. To understand it, this session will discuss materials in:

  1. Dynamic Behaviour of First-Order Systems and Second-Order Systems
  2. Series and Parallel Structures
  3. Recycle Structures
  4. Multi Input-Multi Output Systems

Feedback Loop

This session incdludes materials in:

  1. Process and Instrument Elements of the Feedback Loop
  2. Selecting Controlled and Manipulated Variables
  3. Control Performancce Measures for Common Input Changes

PID Algorithm

PID algorithm is the fundamental of process control. This session includes materials in:

  1. Desired Features of a Feedback Control Algortihm
  2. Block Diagram of the Feedback Loop
  3. Basic Control Mode: Proportional Mode
  4. Basic Control Mode: Integral Mode
  5. Basic Control Mode: Derivative Mode
  6. Basic Control Mode: Proportional-Integral-Derivative Mode
  7. PID Controller

PID Controller Tuning

Tuning is done to get the best response of controller to the actuator. The materials discussed in this material includes:

  1. Defining the Tuning Problem
  2. Determining Good Tuning Constant Values
  3. Correlations for Tuning Constants

Control System Instrumentation

This sessions includes materials in:

  1. Sensors, Transmitters, and Transducers
  2. Final Control Elements
  3. Accuracy in Instrumentation

Stability Analysis and Controller Tuning

Stability analysis will discuss materials in:

  1. Concept of Stability
  2. Stability of Control Systems Principles
  3. Stability of Control Systems: The Bore Method
  4. Controller Tuning Based on Stability: Ziegler-Nichols Close Loop
  5. Controller Tuning and Stability

Process Control Design: Definition and Decisions

This session discuss materials in:

  1. Defining the Design Problem
  2. Measurements and Final Elements
  3. Process Operability
  4. Control Structure, Algortihms, and for Safety
  5. Performance Monitoring

Process Control Design: Managing the Design Procedure

The materials in this session includes:

  1. Defining the Design Problem
  2. Sequence the Design Steps
  3. Temporal Hieraarchy of Control Structure
  4. Process Decomposition
  5. Integrating the Control Design Methods
