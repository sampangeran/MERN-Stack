Chemical Process Simulation

This course dealing with process designs in chemical plants. Topics cover : Process design in chemical plant engineering; reaction path synthesis; reactor selection; separation process selection; process flow sheet and structure synthesis; design of utility systems and heat exchanger networks; The process design incorporates various aspects that needs to be considered, such as technical aspects, economical aspects, safety, and environmental/sustaiability.

References:

  1. Aspen Technology. (2020). Aspen HYSYS Unit Operations: Reference Guide.
  2. Fares, A. D. (2016). Process Simulation Using Aspen HYSYS V8.
  3. Dimian, Alexandre C., Costin S. Bildea, Anton A. Kiss (2014). Integrated Design and Simulation of Chemical Processes. Elsevier

HYSYS Thermodynamics and Streams

This session is an introduction to process simulator, which covers materials in:

  1. Fluid Packages
  2. Hypotheticals
  3. Reactions
  4. Material Stream Properties
  5. Energy Stream Properties

Heat Tranfer Operations

The materials discussed in this session includes:

  1. Air Cooler
  2. Cooler & Heater
  3. Heat Exchanger
  4. Fired Heater
  5. LNG Exchanger
  6. Plate Exchanger

Piping Operations

The materials discussed in this session includes:

  1. Compressible Gas Pipe
  2. Mixer
  3. Pipe Segment
  4. Relief Valve
  5. Tee
  6. Valve

Rotating Equipment

The materials discussed in this session includes:

  1. Expander
  2. Centrifugal Compressor
  3. Reciprocationg Compressor
  4. Pump

Separation Operations

This session discuss materials in:

  1. Component Splitter
  2. Separator, 3-Phase Separator, and Tank
  3. Shortcut Column

Column Operations

This session discuss column in process simulator, including:

  1. HYSYS Column Theory
  2. Column Installation
  3. Column Property View
  4. Column Specific Operations

Solid Separation Operations

This sessions includes materials in:

  1. Simple Solid Separator
  2. Cyclone
  3. Hydrocyclone
  4. Rotary Vacuum Filter
  5. Baghouse Filter

Reactor Operations

This session includes materials in:

  1. Continous-Stirred Tank Reactor (CSTR)
  2. Equilibrium Reactor
  3. Gibbs Reactor
  4. Yield Shift Reactor
  5. Plug Flow Reactor

Logical Operations

This session discuss some logical operation in process simulator, including:

  1. Adjust
  2. Balance
  3. Recycle
  4. Set
  5. Spreadsheet
